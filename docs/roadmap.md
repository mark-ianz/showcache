# Roadmap

This roadmap outlines the future direction of ShowCache, including planned features and known improvements.

## Recently Completed ✅

- **Dynamic Search**: Real-time results across movies, TV, people, and collections.
- **Collection View**: Dedicated landing page for movie franchises and collections.
- **Improved Navigation**: Interactive Hamburger menu, Go Back, and Back to Top buttons.
- **Content Enrichment**: Show facts, facts formatting, and clickable media images.
- **Core UI**: Footer, Error component, and base cinematic styling.

## High Priority (In Progress) 🚀

1. **Authentication**: Allow users to create profiles, sign in, and manage personal watchlists.
2. **Infinite Scrolling**: Replace pagination with a seamless infinite scroll experience on list pages.
3. **Skeleton Loading**: Improve perceived performance with bone-based loading states for cards and text blocks.

## Medium Priority 📈

- **ViewAllMedia Component**: A dedicated gallery view for all images related to a specific show or person.
- **Advanced Filtering**: Filter movies/TV shows by genre, year, rating, and language.
- **User Reviews**: Integration of TMDB user reviews and professional ratings.

## Future Exploration 🔮

- **Social Features**: Sharing favorite collections or movies with friends.
- **Mobile App**: Exploring a native mobile experience using React Native.
- **Offline Support**: PWA (Progressive Web App) functionality for basic browsing without internet.

---

*This roadmap is living documentation and will be updated as the project evolves.*
